# Tutorials (demos)

Each tutorial points to a **runnable directory** under `examples/`. Install the package editable, then follow the README in that folder.

| Tutorial | Demo | What you learn |
|----------|------|----------------|
| [Adaptive difficulty (vigilance-style)](adaptive_difficulty.md) | `examples/adaptive_difficulty_demo/` | Custom policy driving task difficulty; full artifacts + experiment-style logging |
| [Model-based feedback](model_feedback.md) | `examples/model_feedback_demo/` | Custom model + policy mapping neural state to feedback type |
| [PsychoPy end-to-end](psychopy_e2e.md) | `examples/psychopy_e2e/` | Real PsychoPy window, `pop_pending` / `report_realized`, task-level run report |

**Prerequisite:** [Quickstart](../quickstart.md) and [Concepts](../concepts/index.md) (skim).
